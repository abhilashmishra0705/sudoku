This project has been created as a part of the code assignment.


At first, the project needs to be cloned from the github repository.


To run the project using command line, we need to do the following steps: 
1. Go to the repository folder.
2. Clean and install the project using —— “mvn clean install commands”
3. Run the project using               —— “mvn spring-boot:run”
4. The project will then run on localhost:8080


To run the project using IDE(Eclipse), we need to do the following steps: 
1. Import the project as an existing maven project.
2. Clean the project by using 
		Run as --> Maven clean
3. Run the project using
	 	Run as --> Maven Build (Goal = spring-boot:run) --> Run
4. The project will then run on localhost:8080
